# CSCE4350-DatabaseProject
Automobile Company requires a major redesign of their database system. 
ER diagram: [link](https://dbdiagram.io/d/Copy-of-Untitled-Diagram-6909529d6735e1117011a0b8)

# How to use
Install SQL and create a user called 'USER1' with a password 'Password1' and create a database 'cars'.

### Install the driver
`wget https://dev.mysql.com/get/Downloads/Connector-J/mysql-connector-j-8.4.0.tar.gz`

`tar -xzf mysql-connector-j-8.4.0.tar.gz`

### Compile
`javac main.java`

### Run
`java -cp ".:mysql-connector-j-8.4.0/mysql-connector-j-8.4.0.jar" main`
