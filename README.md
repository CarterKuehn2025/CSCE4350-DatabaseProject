## How to use
1. Install SQL and create a user called 'USER1' with a password 'Password1' and create a database 'cars'.

2. Install the driver:
`wget https://dev.mysql.com/get/Downloads/Connector-J/mysql-connector-j-8.4.0.tar.gz && tar -xzf mysql-connector-j-8.4.0.tar.gz`

3. Compile:
`javac main.java`

4. Run:
`java -cp ".:mysql-connector-j-8.4.0/mysql-connector-j-8.4.0.jar" main`
